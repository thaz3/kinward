**Primary action:** Switch Care Recipient. **State label:** Circle-wide context.

**Privacy behavior:** The design never inserts placeholder cards or hidden counts for inaccessible recipients. Generic card errors reveal no protected detail.

**Accessibility annotations:** Context is announced before the page heading; logical landmark and heading order; card actions use full text; no status depends on color; 200% text reflows cards vertically.
